CoinStats Dashboard

Project Title: CoinStats Dashboard

A comprehensive dashboard for visualizing the historical data of various cryptocurrencies including Bitcoin, Ethereum, Bitcoin Cash, Litecoin, and Ripple.

Features:

  • Interactive Date Slider: Select a date range to filter the displayed data.
  • Cryptocurrency Selection: Choose from multiple cryptocurrencies to view specific data.
  • Variable Selection: Display different metrics such as price in dollars, market capitalization, or 24-hour trading volume.
  • Responsive Design: The layout adjusts for optimal viewing on different devices.

Visualization:

  • Dynamic Line Charts: Visualize the selected data over time with smooth transitions.
  • Tooltips: Hover over the chart to see detailed data points with dynamic tooltips.
  • Axes and Labels: Clear and labeled axes for time and selected metrics.

Technology Stack:

  • Frontend: HTML, CSS, JavaScript, D3.js for data visualization.
  • Libraries: Bootstrap for responsive design, jQuery and jQuery UI for interactivity.

How to Use the Website:

  1. Navigate to the Dashboard: Access the CoinStats Dashboard to start exploring cryptocurrency data.
  2. Select Date Range: Use the date slider located at the top of the dashboard to select the range of dates for which you want to view data. The dates will be updated in real-time above the slider.
  3. Choose Cryptocurrency: In the dropdown menu labeled "Cryptocurrency", select the cryptocurrency you are interested in (e.g., Bitcoin, Ethereum).
  4. Select Variable: Use the "Variable" dropdown to choose the data you want to visualize: Price in dollars, Market Capitalization, or 24-hour trading volume.
  5. Interact with the Chart: Hover over the line chart to see detailed information for specific data points. Tooltips will provide precise values.
  6. Observe Dynamic Updates: The chart will automatically update based on your selections, providing a dynamic and interactive experience.
